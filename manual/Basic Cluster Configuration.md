2. **服务器名称设置**  
   点击左下角`系统`，再点击`关于`进入系统信息界面。在`设备名称`里面给服务器主机设置`Master`的名称（注意记住大小写），点击旁边蓝色按钮保存。其他两台子机也按照对应的名称分别设置`Slave1`和`Slave2`。操作如图：
